# Minilo Design System Example App

This example application demonstrates the usage of the `minilo_design_system` package. It includes a widget catalog showcasing all available components in both light and dark modes.

## Getting Started

1.  Ensure you have Flutter installed.
2.  From the root of the repository, navigate to the example directory:
    ```bash
    cd example
    ```
3.  Run the app:
    ```bash
    flutter run
    ```

## Features Demonstrated

- **Theme Switching**: Toggle between light and dark modes to see how components adapt.
- **Component Catalog**: Interactive list of all components (Buttons, Inputs, Cards, etc.).
- **Responsive Layout**: Observe how components behave on different screen sizes.
